# 2023_TheBora_WebDesign
Authors: Bo Kyung Kim, Li Zhi, Bao Qingjun / John Abbott College FSD-10

## Application Url

https://thebora.netlify.app/

## Introduction
The project's focus is to create a visually appealing and user-friendly website for a bubble tea shop using HTML and CSS. The project's main goal is to showcase the bubble tea shop's offerings and brand identity effectively. The website structure includes key sections such as a Homepage, an About Us page, a Menu page, and a Contact page.

## Technology Stack

This web application is built using the following technologies and frameworks:

- HTML
- CSS/Bootstrap

## Features

1. Home Page
- The website will feature an inviting homepage with vibrant images of bubble tea and a welcome message.

2. About Us Page
- The About Us page will delve into the brand's story, values, and founder's inspiration, adding a personal touch to the shop's identity.

3. Menu Page
- The Menu page will present a comprehensive list of flavors, milk teas, fruit teas, and toppings, complete with high-quality images, descriptions, and prices.
- The highlight will be the Product Detail pages, which will provide in-depth information about each bubble tea flavor, including customizable options, nutritional details, and pricing.

4. Contact page
- The Contact page will offer opening hours, location details, and interactive maps, enhancing the shop's accessibility.
